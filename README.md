## 📌 About

Este projeto é uma aplicação de listagem de filmes que apresenta informações detalhadas sobre cada título, incluindo a imagem de capa, o título do filme, uma breve descrição e a nota média representada por estrelas. Os dados dos filmes são obtidos de uma API, garantindo que a lista seja constantemente atualizada com os lançamentos mais recentes. O design do projeto é totalmente responsivo, adaptando-se perfeitamente a diferentes tamanhos de tela, proporcionando uma experiência de navegação intuitiva e agradável em dispositivos móveis, tablets e desktops. O objetivo é oferecer aos usuários uma maneira fácil e eficiente de explorar filmes, com informações visuais e classificações, tudo em uma interface amigável e acessível.

## 🚀 Getting Started

Para rodar este projeto localmente, siga as instruções abaixo:

### Prerequisites

Certifique-se de ter os seguintes pré-requisitos instalados:

- [NodeJS](https://nodejs.org/)
- [Git](https://git-scm.com/)

### Cloning

Clone o repositório:

```bash
git clone https://github.com/DoouglasLobo/listagem-de-filmes
```

### 🚀 Starting

Instale as dependências:

```bash
cd movie-list-app
npm install
```

Execute o projeto:

```bash
npm start
```

A aplicação estará disponível em http://localhost:3000.
